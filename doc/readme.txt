Travel agencies and attractiveness of regions

Main file: travel.pde

The map shows counties of Czech Republic and travel agencies registered in Association of Czech Travel Agents in 2010. Colors of the counties represent number of incoming guests (green component) and number of suicides (red component) in 2010. The color component levels are scaled logarithmically. Each travel agency is represented by a blue circle.

data.txt lists sources of data files used in the visualization.
